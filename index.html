What I Do.

I am highly interested in Software Engineering and Project Management positions. I am a keen learner and like to keep up to date with the latest technological innovations and developments.

  • (Wo)Men2Be

    An Android application that connects girls in high schools from the Urbana-Champaign region to women mentors at the University of Illinois for specific desired technical skills. It pairs mentees with mentors who are experienced and adept at the skillset the mentee wants to learn

  • LecNotes

    Attempted to create an application that takes running notes for lectures by converting speech to text and stores it on the database for easy student access. It sends out push notifications to students if it detects keywords such as "exams", "due dates", etc

  • Data Visualization

    Made a visualization using different sample codes on d3.js to successfully present collected data and answer the question "Does a country's access to electricity and its education expenditure serve as a good representation of where it stands in terms of its Human Development Index (HDI)?" I made a dashboard visualization for the same data using Tableau

  • Tetris

    Made the game Tetris for my ECE 385 final project with my lab partner.The entire game was designed using System Verilog and the DE-2 Board.The USB continuously reads user key presses from the keyboard and sends it to our code in eclipse. Our software code runs the entire game logic and uses the Avalon-MM interface to transfer data from our software to our hardware at each clock cycle

  • Sensors and Circuit

    • PingPongPaddlePro: Created a project that uses various sensors ( Accelerometer, Gyroscope, IMU) to give a ping pong player the angle, velocity and acceleration of his shot so that it could be used as feedback to improve performance
    • PinPoint: Designed a project to detect if a seat is occupied, vacant or broken using pressure sensors and displayed results using LEDs for use in lecture halls
